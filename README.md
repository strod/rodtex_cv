## Local Development

1. Clone the repository:
   ```bash
   git clone git@github.com:strod/rodtex_cv.git
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   ```

3. Visit `http://localhost:8000`
